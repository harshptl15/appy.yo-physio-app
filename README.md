# Download and Setup:

0. Ensure that you have a running IDE (Recommended: [Visual Studio Code](https://code.visualstudio.com/download)) and a database viewing tool (Recommended: [MySQL Workbench](https://dev.mysql.com/downloads/workbench/))
2. Download and Install [node package manager](https://nodejs.org/en/learn/getting-started/an-introduction-to-the-npm-package-manager) if you havent done so already.
3. Open the project within your IDE of choice (Visual Studio Code recommended)
4. Find a terminal within your IDE - At the project root, open it and type: 'npm install' (this will install needed dependencies)
7. in the terminal type: "npm install speakeasy qrcode"
8. Congrats! You can now run the Physio App on your local machine!

## Running The Program

1. In your IDE, navigate to the terminal and run "node index.js"; the response should say "Running on Port 3000" or something similar
2. Now go to an internet browser of your choosing, type in the search bar "localhost:3000", and you should see the sign-in page!
3. Create an account and then log-in once the page redirects you
4. Congrats! You can now roam around the Physio App!

## Included features

- MVC architecture
- Fully developed user interface(UI)
- Access a cloud database using AWS
- Account registrationAccount login
- Account Two-Factor Authentication
- Password hashing for security
- Sessions to control accessibility to the program
- Exercise tracking/favouriting
- EJS for updating views with variables from the controller
- Session logout
